## 支持的三维格式
| 名称 | 格式 | 用途 |
| :-----| :---- | :----|
| Batch 3D model | .b3dm | 地形纹理和地表模型、三维建筑内外表面模型、大规模建筑模型等 |
| Instanced 3D Model | .i3dm | 大量小规模的独立模型，例如树木、路灯、螺栓等|
| Point Cloud | .i3dm | 点云模型 |
| Composite | .cmpt | 多种类型的模型组合格式 |
